# 數位跟蹤狂 (Digital Stalker) - 互動敘事遊戲

這是一個使用 Node.js, Express 和 p5.js 製作的互動敘事遊戲，旨在探討數位足跡、網路隱私與跟蹤騷擾等社會議題。

本專案為「JavaScript 應用程式設計」課程的期末專題。

## 專案介紹

玩家將扮演一名在咖啡廳打工的主角，透過一系列的日常事件，逐步發現自己正被人窺探和跟蹤，並需要在關鍵時刻做出抉擇，導向不同的結局。

## 如何執行

1.  **安裝依賴套件**:
    在專案根目錄下，開啟終端機並執行以下指令：
    ```bash
    npm install
    ```

2.  **啟動伺服器**:
    接著執行：
    ```bash
    npm start
    ```

3.  **開啟遊戲**:
    打開你的網頁瀏覽器，並前往 `http://localhost:3000` 即可開始遊戲。
